【フルスクリーンボタンプラグイン ver1.00】
http://hororo.wp.xdomain.jp/

2021/10/15更新 v510h対応版


-----------------------------------------------------------------------------------------------
■ 概要・機能
-----------------------------------------------------------------------------------------------
　フルスクリーン状態を判別してフルスクリーンボタンの画像を切り替えるプラグインです。
　動画などの右下に付いてるフルスクリーンボタンのイメージです。

　それ以外の機能はありません。


-----------------------------------------------------------------------------------------------
■ プラグインの読み込み方
-----------------------------------------------------------------------------------------------
　(1) ボタン画像は data/image フォルダに格納してください。

　(2) data/others/plugin に 「fullscreen」フォルダを入れてください。

　(3) first.ks （ビルダーは scenario/system/plugin.ks ）でプラグインを呼び出してください。
    　[plugin name=fullscreen]

　(4) [fullscreen] タグで、フルスクリーンボタンを設定します。


-----------------------------------------------------------------------------------------------
■ パラメータ
-----------------------------------------------------------------------------------------------
　◆ [pulugin]・[fullscreen] 用パラメーター（※初期の場合は省略可）

    パラメータ名   値             初期値                説明
    img            画像名         fullscreen.png        フルスクリーン画像
    img_exit       画像名         fullscreen_exit.png   フルスクリーン解除画像
    zindex         数値           99999999              メニューなどの上に表示したい時は 9999999999
　　clearfix       yes/no         yes                   noにするとclearfixタグで削除されません。

　・ [pulugin] で指定した値は [fullscreen] で省略できます。

　・ [fullscreen] タグには、[button] タグのパラメータがそのまま使えます。
　　 ※但し、enterimg を除く

　・ y・x は、マイナス値で指定する事で、bottom・right になります。
　　 例：y=-10 x=-10 で右下 10px 位置に配置されます。

　・ y・x の初期値は -10 です。

　・ width・height の初期値は 50 です。


-----------------------------------------------------------------------------------------------
■ 記述例
-----------------------------------------------------------------------------------------------
　◆ボタン画像を指定する
　　[fullscreen img=fullscreen.png img_exit=fullscreen_exit.png]

　◆メニュー画面などでも表示する
　　[fullscreen zindex=9999999999]

　◆[clearfix] で削除しないようにする
　　[fullscreen clearfix=no]

　　※削除したいときは [free name="fullscreen" layer="fix"] で削除できます。


-----------------------------------------------------------------------------------------------
■ style.css、class名について
-----------------------------------------------------------------------------------------------
　・同梱のサンプル画像（黒画像）に合わせて、opacity で透明度を変えています。
　　フルスクリーン時は、完全透明になるようにしています。

　・スマホの時は、ボタンに 「mobile」 が追加されます。

　・clearfix=no 時は、「fixlayer」を削除し、「fix」が追加されます。


-----------------------------------------------------------------------------------------------
■ その他
-----------------------------------------------------------------------------------------------
　・デザイン変更は画像を差し替えてください。
    画像は image フォルダに保存してください。

　・enterimg は非対応です。マウスオーバー時の画像差し替えはできません。
　　CSSで指定してください。
